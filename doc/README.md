## 📊 项目概览

### 🌟 核心特性

- 🌙 **匿名树洞** - 安全的匿名分享空间
- 🔗 **WordPress集成** - 支持内容转发到WordPress站点
- 🎨 **现代UI** - 基于玻璃拟态的美观界面
- 🔐 **用户认证** - 完整的注册登录系统
- 🐳 **容器化部署** - 一键Docker部署

### 🏗️ 技术栈

**后端**
- Go 1.21 + Gin
- MySQL 8.0 + Redis
- GORM + JWT

**前端**
- React 18 + TypeScript
- Vite + Tailwind CSS
- Framer Motion + Axios

**部署**
- Docker + Docker Compose
- Nginx + 反向代理
